Heads Up (But not copyrighted)

This project is essentially a quiz on the "Processing" application that the user can take to find out their "personality" type. The quiz questions themselves are pretty inappropraite, but a lot of work was done to make them and make them work properly. I ended with more than 5 questions on the quiz as well as 3 possible personality types. This project was fun despite its rigorous work and it helped me figure out several things like how to use end conditions to let the code choose the personality type for the quiz taker. Overall, it was a very fun project and it helped me develop more useful skills than anything.
